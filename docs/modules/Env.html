Env module

Information about Elona foobar and mod engine environment.

Usage

local Env = Elona.require("Env")

Fields

LUA_VERSION Lua version.
ELONA_VERSION Elona version.
ELONA_FOOBAR_VERSION Elona foobar version.
MOD_API_VERSION Mod API Version.

Fields

# LUA_VERSION

Lua version.

E.g., "5.3". Note that it is different from "_VERSION", one of the Lua built-in global constants.

  • LUA_VERSION : (string)
# ELONA_VERSION

Elona version.

Currently, it is always "1.22".

  • ELONA_VERSION : (string)
# ELONA_FOOBAR_VERSION

Elona foobar version.

E.g., "1.2.3"

  • ELONA_FOOBAR_VERSION : (string)
# MOD_API_VERSION

Mod API Version.

WARNING: it always returns "0.1" so far, and not incremented every release until the mod API is stabilized.

  • MOD_API_VERSION : (string)
