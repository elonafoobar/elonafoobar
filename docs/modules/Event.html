Event module

Allows you to register new event callbacks when initializing mods.

See also

  • Event.EventKind

local Event = Elona.require("Event")

Functions

# register (event_id, handler)

Registers a new event handler.

This function will append a new callback hander for the given event to the end of the callback list for that event. Then, when the event is fired, each callback in the list will be run in the order in which they were registered. The callbacks will receive the arguments specified in the documentation for the event. These arguments will differ between event types. Read the documentation for the event you're handling to see the available event arguments for it.

Parameters:
  • event_id : (Event.EventKind) the event kind to register
  • handler : (function) the handler for this event
Usage: 
local GUI = Elona.require("GUI")
local Event = Elona.require("Event")
local function my_chara_moved_handler(chara)
    GUI.txt("The character " .. chara.name .. " is about to move. ")
end

Event.register(Event.EventKind.CharaMoved, my_chara_moved_handler)
# trigger (event, data)

Triggers an event from a Lua script.

  • event : (Event.EventKind) the event to trigger
  • data : (table) the event data
